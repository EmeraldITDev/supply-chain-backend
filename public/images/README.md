# Company Logo for Purchase Orders

## Logo Placement

Place your company logo file in this directory (`public/images/`) with one of the following names:

- `logo.png`
- `logo.jpg`
- `company-logo.png`
- `company-logo.jpg`
- `emerald-logo.png`
- `emerald-logo.jpg`

## Recommended Format

- **Format**: PNG or JPG
- **Size**: Recommended 240x120px (will be scaled to 120x60px in PDF)
- **Background**: Transparent PNG preferred, or match your PDF background

## Logo Description

The logo should be the Emerald Industrial Co. CFZE logo with:
- Stylized "E" letter in dark teal/deep blue
- "EMERALD" text in lighter teal/turquoise
- "INDUSTRIAL CO.," text in dark teal
- "CFZE" text in lighter teal

Once the logo file is placed in this directory, it will automatically appear in all generated Purchase Order PDFs.
